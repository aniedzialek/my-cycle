# my-cycle

Install angular-cli:
```bash
npm install -g angular-cli
```
Clone github repository to your local machine:
```bash
git clone https://github.com/aniedzialek/my-cycle.git
```
Navigate to repository directory:
```bash
cd my-cycle
```
Install modules:
```bash
npm install
```
In your browser open:
```bash
http://localhost:4200/
```
Start app:
```bash
npm start
```
